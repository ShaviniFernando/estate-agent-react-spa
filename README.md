# Estate Agent React SPA

A responsive single-page application built with React and Vite for browsing estate agent property listings. It includes dynamic filtering, drag-and-drop favourites functionality, and detailed property views with interactive tabs and Google Maps integration.

## Tech Stack
- **Framework:** React + Vite
- **Routing:** react-router-dom
- **UI Components:** react-select, rc-slider, react-datepicker, react-tabs
- **Testing:** Jest

## Setup Instructions

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the local development server:**
   ```bash
   npm run dev
   ```

3. **Run the test suite:**
   ```bash
   npm test
   ```

4. **Deploy to GitHub Pages:**
   ```bash
   npm run deploy
   ```
